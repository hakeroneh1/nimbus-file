# ☁️ Nimbus File

### Безлимитное облачное хранилище на базе Telegram

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Python](https://img.shields.io/badge/python-3.9%2B-orange)

Nimbus File — это мощное и гибкое облачное хранилище, которое использует Telegram как бэкенд. Любые файлы любого размера автоматически разделяются на чанки и распределяются по вашим Telegram каналам через ботов. Никаких ограничений по размеру и количеству файлов.

---

## ✨ Возможности

| | Функция | Описание |
|---|--------|----------|
| 📁 | **Безлимитное хранилище** | Используй Telegram как бесплатный бэкенд |
| 🚀 | **Балансировка нагрузки** | Автоматическое распределение между ботами и каналами |
| 🔐 | **Безопасность** | JWT аутентификация, только вы имеете доступ |
| 🌓 | **Темная тема** | Автоматически сохраняется в cookie |
| 📱 | **Адаптивный дизайн** | Работает на ПК, планшетах и телефонах |
| 🔑 | **API ключи** | Для интеграции с другими сервисами |
| 📤 | **Шеринг файлов** | Делись файлами по публичной ссылке |
| 📁 | **Папки** | Организуй файлы как хочешь |
| ✏️ | **Редактирование** | Переименовывай, удаляй, перемещай |
| 📊 | **Прогресс загрузки** | Визуальный индикатор для больших файлов |
| ⚡ | **Молниеносно** | Асинхронная обработка, нет блокировок |

## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонируем репозиторий
git clone https://github.com/yourusername/nimbus-file.git
cd nimbus-file

# Устанавливаем зависимости
pip install -r requirements.txt
```
